# Multisender Toolkit

## One-to-Multi Transfer <a href="#one-to-multi" id="one-to-multi"></a>

**CiaoTool One-to-Multi Transfer** is an automated distribution tool designed for efficient, large-scale token transfers. It enables a single wallet to send tokens to an unlimited number of recipient addresses in one operation, eliminating the tedious and time-consuming process of manual individual transfers.

The tool supports both automated custom allocations per address and global equal distribution, making it ideal for high-frequency, large-scale asset scheduling scenarios such as project airdrops, community rewards, team payments, and multi-wallet fund positioning.

## Multi-to-One Transfer

**CiaoTool Multi-to-One Transfer** is an automated fund consolidation tool designed for efficient asset collection and management. It allows multiple independent source wallets to be deployed simultaneously, seamlessly gathering fragmented tokens from each wallet into a single designated destination wallet.

After multi-address market-making activities or airdrop campaigns, assets are often left spread across many wallets. This tool completely eliminates the tedious process of switching between accounts and manually approving repeated transactions.

## Multi-to-Multi Transfer <a href="#multi-to-multi" id="multi-to-multi"></a>

**CiaoTool Multi-to-Multi Transfer** is an advanced tool for complex matrix-style fund routing. It enables importing multiple senders and receivers simultaneously, triggering parallel independent transfer pathways in a single operation.

Compared to traditional back-and-forth streams, this model perfectly fits complex asset restructuring, offering granular control over fund flows and specific amounts for each account.

## Relay Transfer <a href="#relay-transfer" id="relay-transfer"></a>

**CiaoTool Relay Transfer** is a privacy-focused routing tool designed to enhance on-chain asset migration and obscure fund flows. Instead of sending assets directly, the transfer is routed through newly created, trace-free intermediary wallets.

Its key advantage is breaking the direct linkage between addresses. Combined with customizable time delays and multi-path distribution, it effectively cuts off the direct correlation on the public ledger while optimizing token distribution structures.
