# How to Get a Token Logo Displayed?

## Why Tokens Don’t Have Logos

* On EVM Network like BSC and Ethereum, a token is just a smart contract. The chain doesn’t store logos, descriptions, websites, or social links.
* All token logos you see in wallets, exchanges, or data platforms are **metadata maintained by third-party platforms**.

**Platforms add logos to:**

* Improve readability and recognition
* Reduce user errors
* Increase project professionalism and credibility

**To show a logo, you must apply on each platform individually.**

## From Basic Display to Platform Recognition

* Instead of rushing to CoinMarketCap (CMC) or CoinGecko (CG), follow a step-by-step approach based on **visibility priority + approval likelihood**
* **Common sequence:** Wallets / Info Platforms → Block Explorers → Data Aggregators → Auto-fetch by DEXs。

* **Requirements:** valid website, social links, ongoing activity
* **Success rate:** low

## CMC / CG

[**CoinMarketCap**](https://coinmarketcap.com/) **(CMC) and** [**CoinGecko**](https://www.coingecko.com/) **(CG):** Global authority platforms

* If token is listed, logo syncs, wallets and DEXs can fetch it, boosts credibility
* Very strict approval, better for tokens with trading volume and user base
* **Apply:** official forms or [**Contact Us**](https://t.me/ciaotool)
* **Success rate:** low

{% hint style="warning" %}
**Note:** As the listing fees on these two platforms are relatively high, be sure to submit your application only through official or trusted channels to avoid scams.
{% endhint %}

## PancakeSwap / Uniswap Update <a href="#pancakeswap-uniswap-update" id="pancakeswap-uniswap-update"></a>

* **No direct submission channel**
* **Logo display depends on:**
  1. Wallet/platform listing
  2. Data aggregator listing (CMC/CG)
  3. Indirect DEX aggregation (DEXTools, project updates)

## MetaMask Logo Update

#### **⬆️ Wait for official data indexing or manual updates**

* If the token has already been listed on major platforms (**such as CG, CMC, Etherscan, etc.**), simply wait for the update to take effect.
* If it has not been updated yet, users can use the one-click add feature on these platforms to automatically add the token and its official logo to their wallet, then confirm the wallet pop-up.

{% hint style="info" %}
**Note:** Manual updates are only available if the token has already been listed on major platforms.
{% endhint %}
