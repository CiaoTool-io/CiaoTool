## What Is CiaoTool Robinhood Chain Add V3 Liquidity?

<figure><picture><source srcset="../../../../.gitbook/assets/ScreenShot_2026-07-22_155435_188.png" media="(prefers-color-scheme: dark)"><img src="../../../../.gitbook/assets/image (1092).png" alt=""></picture><figcaption></figcaption></figure>

CiaoTool Add V3 Liquidity is a no-code tool for adding concentrated liquidity to an existing Uniswap V3 pool on Robinhood Chain.

Unlike V2, where liquidity is distributed across the entire price curve, Uniswap V3 allows liquidity providers to deploy assets within a selected minimum and maximum price range. Liquidity is active and may earn trading fees only while the pool price remains within that range.

CiaoTool simplifies the required V3 smart-contract interactions into one interface. Users enter the existing V3 Pool Address, verify the token pair and fee tier, configure a price range, enter the required asset amounts, complete token approvals, and submit the liquidity transaction without writing code.

### Set Price Range

<figure><img src="../../../../.gitbook/assets/image (1096).png" alt=""><figcaption></figcaption></figure>

1. Range size determines the level of yield (applicable to both LP fee yields and CAKE yields):

* The narrower it is, the higher the yield, and the greater the impermanent loss;
* The wider it is, the lower the yield, and the smaller the impermanent loss;
* A full-range setting operates similarly to V2, and the yield will be very small.

2. When out of range:

* The V3 pool generates no yield;
* The liquidity position will turn into a single token;
* You can remove liquidity and add it again, or wait for the price to return to the range;
* You can add a V2 liquidity pool to ensure base liquidity outside the V3 range.

3. Single-Sided Rule:

* Single-sided deposits on Uniswap/Pancake V3 only apply when the current price is outside your designated range. Full-range mode cannot be single-sided.
{% endstep %}

## FAQs

<details>

<summary><strong>What is Add V3 Liquidity?</strong></summary>

Add V3 Liquidity means creating a new concentrated-liquidity position in an existing Uniswap V3 pool within a selected price range.

</details>

<details>

<summary><strong>Does Add V3 Liquidity create a new pool?</strong></summary>

No. It uses an existing, initialized V3 pool. Use Create V3 Liquidity Pool if the required token-pair and fee-tier pool does not exist.

</details>

<details>

<summary><strong>Do I always need to provide both tokens?</strong></summary>

No. If the selected range includes the current price, both assets are generally required. If the range is entirely above or below the current price, only one asset may be required.

</details>

<details>

<summary><strong>What happens when the price moves outside my range?</strong></summary>

The position becomes inactive and stops earning swap fees until the price returns to the configured range.

</details>

<details>

<summary><strong>Can I change the price range later?</strong></summary>

An existing V3 position’s price range cannot normally be edited directly. To use a different range, liquidity must generally be removed and a new position created.

</details>

<details>

<summary><strong>How are V3 trading fees received?</strong></summary>

Trading fees accrue to the position while it is active. They are not necessarily transferred automatically and may need to be collected through a compatible V3 position-management function.

</details>

<details>

<summary><strong>How does CiaoTool protect the imported private key?</strong></summary>

The private key is processed locally in the browser and is not uploaded, transmitted, stored, logged, or written to Local Storage. Imported private-key data is cleared when the page is closed or refreshed.

</details>
