### 🧩如何在RobotPump创建并买入？

#### **1.连接钱包**

* 点击右上角【连接钱包】，选择钱包地址并确认授权。
* 页面右上角显示钱包地址即表示绑定成功。

<figure><img src="../../.gitbook/assets/image (254).png" alt=""><figcaption></figcaption></figure>

#### 2.**填写代币基础信息** <a href="#di-er-bu-tian-jia-she-jiao-lian-jie-ke-xuan-dan-qiang-lie-jian-yi-tian-xie" id="di-er-bu-tian-jia-she-jiao-lian-jie-ke-xuan-dan-qiang-lie-jian-yi-tian-xie"></a>

* **私钥：**&#x8F93;入用于发币的控制地址私钥。注意安全，建议使用测试钱包或专用地址
* **钱包地址 & 余额：**&#x7CFB;统自动解析私钥生成钱包地址，并显示当前 IOTX 余额
* **Dev买入：**&#x8BBE;置Dev买入数值
* **代币名称：**&#x54;oken 的完整名称（如：CiaoTool ）
* **代币简称：**&#x54;oken 符号（如：CIAO）
* **描述：**&#x7B80;要介绍代币用途、故事或 Meme 背景（选填）
* **上传图片：**&#x4E0A;传 Token 图标，建议 1000×1000 像素，支持 jpg/png/webp/gif
* **添加社交链接**：虽为可选项但强烈建议填写，**官网、X / Twitter、Telegram**

<figure><img src="../../.gitbook/assets/image (251).png" alt=""><figcaption></figcaption></figure>

#### 3. 配置内盘买入参数 <a href="#di-san-bu-pei-zhi-nei-pan-mai-ru-can-shu" id="di-san-bu-pei-zhi-nei-pan-mai-ru-can-shu"></a>

* 内盘买入意味着：**你提前布局，控制初期价格走势**，在图表上线前通过脚本抢先买入。
* **点击导入私钥**，批量添加最多 20 个钱包地址
* 设置 买入金额

<figure><img src="../../.gitbook/assets/image (252).png" alt=""><figcaption></figcaption></figure>

#### 4. 配置外盘买入参数 <a href="#di-si-bu-pei-zhi-wai-pan-mai-ru-can-shu" id="di-si-bu-pei-zhi-wai-pan-mai-ru-can-shu"></a>

* 这是此功能的核心亮点，允许你批量导入多个地址（最多5个地址），在代币创建的瞬间 **「自动完成买入」**，实现抢跑保护 + 初始建仓。

<figure><img src="../../.gitbook/assets/image (253).png" alt=""><figcaption></figcaption></figure>

**配置方式：**

1. 点击“导入私钥”，支持批量粘贴多个私钥
2. 系统自动读取每个私钥的 IOTX 余额
3. 为每个地址设置「买入金额」 ，单个地址无上限买入金额，刷新余额查看即可
4. 可选择刷新余额或调整买入比例
5. 系统将展示每个地址的状态 + 操作按钮

{% hint style="info" %}
买入地址的IOTX余额必须大于 1！
{% endhint %}

#### 5. 点击“创建并买入” <a href="#di-wu-bu-dian-ji-yi-jian-chuang-jian-bing-mai-ru" id="di-wu-bu-dian-ji-yi-jian-chuang-jian-bing-mai-ru"></a>

* 完成所有信息后，点击页面底部按钮

#### ❓ 常见问题 FAQ <a href="#chang-jian-wen-ti-faq" id="chang-jian-wen-ti-faq"></a>

**1. IoTeX 创建代币后多久可以交易？**

* 答：创建后代币会立即上线，完成买入后即可在链上自由流通。

**2. 买入代币的钱包可以设置多个吗？**

* 答：可以，可添加多个钱包进行同步买入，适用于控盘或分发策略。

**3. 如何查看创建成功的代币？**

* 答：您可在交易成功后跳转查看代币合约地址，或在 IoTeX 区块浏览器查询。

**4. 创建失败会扣 Gas 吗？**

* 答：若交易未成功，链上仍可能产生一定的 Gas 消耗，请确保钱包余额充足
